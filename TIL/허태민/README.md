### 4. 예전 희찬이 아이디어 - 노래 허밍 지식인.
    - 음은 알겠는데 무슨 노래일지 기억이 안날 때, 올릴 수 있는 사이트

    - 오랫동안 못 맞추면, 챌린지 느낌으로 메인에 띄워도 될 것 같음
    - 또 내공처럼 포인트를 모아 랭킹을 하고, 분야별 랭킹 기반으로 자격증처럼 만들어서 주면 재밌을 것 같음 (ex. 걸그룹 정보 정복자)
    - 목소리 올리기 민망한 사람들을 위해 음성변조 기능 넣으면 좋을 듯
